# 移动网页目录管理应用程序
	##2年前做的，可作为相关技术框架的练习实践样例，理解如何创建一个跨设备应用程序。
	* 前端使用了 ionic+AngularJS+leancloud 组合，其中bass管理系统leancloud与parse一致，虽然parse
	* 后台管理使用了 Bootstrap+AngularJS+leancloud，一页式Web APP。
